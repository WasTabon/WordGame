# WordGame — Итерация 1: Фундамент + Главное меню

## Что сделано в этой итерации

Заложен фундамент проекта и построено главное меню со всеми попапами. Геймплея пока нет — его начнём в итерации 2.

### Менеджеры (DontDestroyOnLoad)
- **SoundManager** — управление музыкой и SFX, громкость сохраняется в PlayerPrefs (`WG_MusicVolume`, `WG_SfxVolume`)
- **SceneLoader** — плавный fade-переход между сценами (готов, но ещё не используется — пригодится в итерации 2)

### UI
- Главное меню: заголовок **WORD GAME**, большая кнопка **PLAY**, внизу **SETTINGS** и **HOW TO PLAY**
- **SettingsPopup** — два слайдера (Music / SFX), значения сохраняются между запусками
- **HowToPlayPopup** — статичный текст с правилами игры
- **ModeSelectPopup** — выбор режима (ESCAPE / EXPLORE), сохраняет выбор в `GameMode.Current`
- **PlaceholderPopup** — заглушка, показывающая выбранный режим ("Coming in next iteration")

### Анимации (DOTween)
- Вход в главное меню — заголовок и кнопки поочерёдно появляются с `OutBack`
- Заголовок покачивается вверх-вниз (yoyo loop)
- Все попапы — fade + scale-in с `OutBack`, обратно — fade + scale-out с `InQuad`
- Все кнопки имеют press-feedback (`UIButtonFeedback`) — сжимаются при нажатии

## Установка и запуск

### Требования
- Unity **2022.3.62f** (или совместимый 2022.3 LTS)
- 2D template, Built-in Render Pipeline
- Платформа: **iOS**, ориентация Portrait

### Шаги

1. **Создать новый Unity-проект** (2D, Built-in).

2. **Импортировать зависимости ДО запуска editor-скрипта:**
   - В Package Manager установить **TextMeshPro** (обычно уже есть), при первом обращении согласиться на импорт **TMP Essentials**
   - Из Asset Store импортировать **DOTween (HOTween v2)** — бесплатную версию
   - После импорта DOTween в меню: `Tools > Demigiant > DOTween Utility Panel > Setup DOTween...`

3. **Скопировать содержимое архива в проект:**
   - Распаковать архив и положить папку `Assets/WordGame/` в `Assets/` проекта

4. **Запустить editor-скрипт:**
   - В меню Unity: `WordGame > Setup Main Menu (Iteration 1)`
   - Скрипт создаст сцену `Assets/WordGame/Scenes/MainMenu.unity`, построит иерархию, пропишет все ссылки и сохранит сцену
   - Сцена будет автоматически добавлена в Build Settings

5. **Нажать Play** — откроется главное меню.

### Настройка под iOS Portrait
В `Edit > Project Settings > Player > iOS`:
- Default Orientation: **Portrait**
- Auto Rotation: отключено (Portrait only)

В Game-вкладке выставить разрешение **1080 × 1920 Portrait** для превью.

## Как тестировать

| Проверка | Ожидаемое поведение |
|---|---|
| Запуск сцены | Плавная анимация появления меню, заголовок покачивается |
| Нажатие на любую кнопку | Кнопка сжимается при нажатии, разжимается при отпускании |
| PLAY | Открывается ModeSelectPopup с двумя вариантами |
| ESCAPE / EXPLORE | ModeSelectPopup закрывается, открывается PlaceholderPopup с выбранным режимом |
| SETTINGS | Открывается попап, слайдеры двигаются, значения в процентах обновляются |
| Settings → закрыть → снова открыть | Значения слайдеров восстанавливаются (PlayerPrefs работают) |
| Выход из Play и повторный запуск | Громкость остаётся такой, как выставили |
| HOW TO PLAY | Открывается попап с текстом правил |
| Клик на затемнённый фон попапа | Попап закрывается |
| Клик на крестик (×) в углу попапа | Попап закрывается |

## Известные моменты

- Звуковые клипы (`buttonClickClip`, `popupOpenClip`, `popupCloseClip`) не назначены — SoundManager это корректно обрабатывает (тишина). Можно кинуть в слоты в инспекторе свои клипы, и звук появится сразу
- SceneLoader присутствует в сцене, но пока не вызывается — используем в итерации 2 при переходе MainMenu → Game

## Замечания по идемпотентности

Editor-скрипт можно запускать повторно — он ничего не дублирует:
- Ищет существующие объекты по имени (`FindOrCreateRoot` / `FindOrCreateChild`)
- Компоненты добавляет только если их нет (`GetOrAdd<T>`)
- Обновляет значения (цвета, позиции, ссылки) каждый раз

Если вручную удалишь что-то в сцене между итерациями — в следующих editor-скриптах этого не будет трогаться, потому что каждая итерация настраивает только свои объекты.
